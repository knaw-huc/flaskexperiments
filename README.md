# Flask experiments

At least needed for eCodices. Minimal examples, cors, jinja, links, tutorials

The first time:
```
cd repository
virtualenv env
source env/bin/activate
pip3 install flask
pip3 install flask-cors
python3 app.py 
```

Later:
```
cd repostory
source env/bin/activate
python3 app.py # flask server starts
```
